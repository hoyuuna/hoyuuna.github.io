Tài liệu server Minecraft

Chào mừng

  • Chào mừng bạn đến với tài liệu server Minecraft Hoxilo Smol House (Căn nhà nhỏ của Hoxilo)
  • Vui lòng đọc tài liệu thật cẩn trọng để tránh sự cố đáng tiếc khi chơi!

Hướng dẫn tải bằng văn bản

  • Chuyển vùng App Store từ Việt Nam qua Singapore trong cài đặt tài khoản Apple
  • Tải Blue Archive (không vào game)
  • Tải Turbo VPN hoặc bất kì phần mềm VPN nào khác
  • Kích hoạt VPN và vào game tải 400MB file đầu game
  • Nếu game hiện Service is not supported in this region mà không có icon X thì nhấn vào khoảng không góc trên bên phải màn hình
  • Đăng nhập, sau khi đã đăng nhập xong tắt VPN
  • Tải tiếp 10GB file game và vào chơi
