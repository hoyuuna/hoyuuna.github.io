Yuuka GPT Bot

Yuuka GPT Gemini Discord Bot

1. Tải dự án

  • Link Github của dự án có thể được tìm thấy tại đây
  • Sử dụng nền tảng Python 3.11 (đã được tích hợp sẵn Secret shortcut cho Replit)
  • Lưu ý cài đặt các gói cần thiết trong file requirements.txt

2. Các gói yêu cầu

  • aiohttp
  • discord.py
  • google-generativeai
  • python-dotenv
  • Nếu SD hosting nhập requirements.txt vào mục Requirements file

3. Environment

  • Vui lòng chỉnh sửa file .env trước khi khởi động bot!
  • Thay DISCORD_BOT_TOKEN bằng token bot của bạn (https://discord.com/developers/)
  • Thay GEMINI_TOKEN bằng API Gemini (https://aistudio.google.com/apikey)
  • Dòng MAX_HISTORY vui lòng để mặc định nếu bạn không hiểu!

4. Model AI

  • Vào file gasmii.py để chỉnh sửa model
  • Thay giá trị model_name bằng model bạn muốn SD (VD: gemini-2.0-flash-exp)
  • Thay giá trị temperature bằng giá trị từ 0-2 (giá trịcàng cao độ sáng tạo càng cao)
