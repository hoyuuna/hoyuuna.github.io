Hoyuuna

Yuuka GPT Gemini Discord Bot

1. Tải/clone dự án

  • Mã nguồn của dự án có thể được tìm thấy tại đây
  • Sử dụng nền tảng Python 3.11
  • Lưu ý cài đặt các gói cần thiết trong file requirements.txt

2. Các gói yêu cầu

  • aiohttp
  • discord.py
  • google-generativeai
  • python-dotenv
  • Nếu SD hosting nhập requirements.txt vào mục Requirements file để tự động cài

3. Environment

  • Vui lòng chỉnh sửa file .env trước khi khởi động bot!
  • Thay DISCORD_BOT_TOKEN bằng token bot của bạn (https://discord.com/developers/)
  • Thay GEMINI_TOKEN bằng API Gemini (https://aistudio.google.com/apikey)
  • Giá trị MAX_HISTORY là số tin nhắn tối đa bot ghi nhớ vào lịch sử. Mặc định 15, để 0 để tắt
  • Với Replit sử dụng Secret, xóa file .env

4. Tùy biến AI

  • Vào file gasmii.py để chỉnh sửa model
  • Thay giá trị model_name bằng model bạn muốn SD (VD: gemini-2.0-flash-exp) (xem các model có thể sử dụng tại mục 4.1)
  • Thay giá trị temperature bằng giá trị từ 0-2 (giá trị càng cao độ sáng tạo càng điên rồ, độ chính xác càng thấp)

4.1. Model AI

  • Gemini 2.0 Flash Experimental: gemini-2.0-flash-exp Nên dùng
  • Gemini 1.5 Pro: gemini-1.5-pro
  • Gemini 1.5 Flash: gemini-1.5-flash
  • Gemini 1.5 Flash-8B: gemini-1.5-flash-8b
