Hoyuuna

Ayane MCSV Status Bot

1. Ayane là gì?

    Ayane là bot Discord giúp bạn theo dõi trạng thái mọi server Minecraft chính xác hoàn toàn miễn phí! Ngoài ra bot còn cung cấp các thông tin như danh sách người chơi, phiên bản server v..v..

2. Mời bot

    Nhấn vào đây để mời bot vào server của bạn

3. Lệnh

  • /status
  • /support
  • Lệnh /status có tác dụng để tra trạng thái server Minecraft (xin lưu ý là trạng thái sẽ không được cập nhật thời gian thực)
  • Lệnh /support có tác dụng để báo cáo sự cố và mời bot vào server
    Lệnh /status có 3 nhánh con:
  • Nhánh type sẽ cho người dùng chọn giữa 2 loại server: Java (PC) hoặc Bedrock (BR/PE)
  • Nhánh ip sẽ cho người dùng nhập địa chỉ IP của server cần tra cứu (VD: hoyuuna.mc.sv)
  • Nhánh port sẽ cho người dùng nhập cổng (port) của server cần tra cứu (VD: 13245) - Cần thiết khi server có cổng!
